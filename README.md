🎬 Netflix Data Analysis Project
📌 Project Overview

This project focuses on analyzing the Netflix Movies & TV Shows dataset to extract meaningful business insights using Python and data analytics techniques.

The goal of this project is to perform data cleaning, exploratory data analysis (EDA), and visualization to understand content trends, audience preferences, and platform growth patterns.

This project demonstrates strong skills in:

Data Cleaning

Data Manipulation

Data Visualization

Business Insight Generation

🛠️ Tools & Technologies Used

Python

Pandas – Data cleaning & transformation

NumPy – Numerical operations

Matplotlib & Seaborn – Data visualization

Jupyter Notebook

📊 Key Analysis Performed

✔️ Movies vs TV Shows distribution
✔️ Content added over the years
✔️ Top countries producing Netflix content
✔️ Most popular genres
✔️ Rating distribution (TV-MA, PG, etc.)
✔️ Year-wise content growth trend
✔️ Duration analysis (Movie minutes & TV seasons)

🔎 Data Cleaning Steps

Handled missing values

Converted date columns to proper datetime format

Removed duplicates

Standardized text columns

Extracted useful features (Year, Month, etc.)

📈 Key Insights

Netflix has significantly increased content production after 2015.

Movies dominate the platform compared to TV Shows.

The United States contributes the highest amount of content.

Drama and International content are among the most frequent categories.

🎯 Business Impact

This analysis helps:

Understand content investment trends

Identify dominant genres and countries

Support data-driven decisions for content strategy
