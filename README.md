🎬 NETFLIX DATA ANALYSIS PROJECT



🚀 PROJECT OVERVIEW

This project analyzes the Netflix Movies & TV Shows dataset using Python to extract meaningful business insights through data cleaning, exploration, and visualization.

The main objective is to understand content trends, growth patterns, and audience preferences using real-world data.

🛠️ TOOLS & TECHNOLOGIES

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

📊 KEY ANALYSIS PERFORMED

✔️ Movies vs TV Shows Distribution
✔️ Year-wise Content Growth
✔️ Top Content Producing Countries
✔️ Most Popular Genres
✔️ Ratings Distribution
✔️ Duration Analysis
✔️ Monthly & Yearly Trends

🧹 DATA CLEANING PROCESS

Handled missing values

Removed duplicates

Converted date columns to datetime

Standardized categorical columns

Extracted year and month features

📈 KEY INSIGHTS

Rapid content growth after 2015

Movies dominate the platform

United States produces the highest content

Drama & International genres are most common
